**Lodz Badminton Weekend Friendly Tournaments**
A website to conduct doubles tournament between advanced and intermediate players. 
The application shuffles based on skill level and ensures no group is strong or weak. A total friendly application designed for friends based on our weekly requirement
Group 1: Set of Advanced Players
Group 2: Set of Intermediate Players
Doubles Partners are a mix of one advanced and one intermediate
Each time a tournament is initiated the same set of players are not partners again
